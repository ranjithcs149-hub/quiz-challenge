## üìù Game Description
Quiz Challenge is an interactive browser-based quiz game developed using HTML, CSS, and JavaScript.  
The game presents multiple-choice questions to the player, who must answer them within a limited time.  
Each correct answer increases the player‚Äôs score. The game tests both knowledge and speed, making it engaging and challenging.  
The game ends when the timer runs out or when all questions are answered.

---

## ‚ñ∂Ô∏è How to Play
1. Open the game in a web browser.
2. Click the **Start Quiz** button on the start screen.
3. A question with multiple-choice options will be displayed.
4. Select one option as your answer.
5. If the answer is correct, your score increases.
6. The timer continues to count down during the quiz.
7. The game ends when the timer reaches zero or all questions are completed.
8. The final score is displayed on the Game Over screen.
9. Click **Restart** to play the game again.

## üß† Programming Concepts Used

### Variables & Data Types
Used to store score, timer value, game state, and current question index.

### Operators
Arithmetic operators for score calculation, relational and logical operators for game control.

### Conditional Statements
if / else statements used to check answers, timer completion, and game state transitions.

### Loops
- for loop to display multiple-choice options.
- setInterval loop for the countdown timer.

### Functions
- startGame()
- showQuestion()
- checkAnswer()
- startTimer()
- endGame()
- restartGame()

### Arrays & Objects
Questions stored as an array of objects containing the question, options, and correct answer.

### DOM Manipulation
Dynamic updates of questions, score, timer, and screen visibility using JavaScript.

### Event Handling
Button click events for starting the quiz, selecting answers, and restarting the game.
